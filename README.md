# 투약 - 의약품 복용 이력 관리 및 안전정보를 한눈에!

## 서비스 소개

- 서비스 이용자는 자신의 처방전을 사진찍음으로서 의약품 복용 이력을 관리할 수 있고 해당 의약품에 대한 안전정보를 제공받는다. 또한 자신이 복용하고 있는 의약품과 상성이 맞는 혹은 맞지 않는 건강기능식품 정보를 제공받을 수 있고 상성이 맞는 건강기능식품을 주문할 수 있다.
